# Theoicode Mastery App

This is a Vite + React + TypeScript application bootstrapped with shadcn-ui and TailwindCSS. It serves as the frontend for the Theoicode Mastery curriculum.

## Getting Started

To work on the project locally, make sure you have Node.js and npm installed.

```sh
# clone the repository and install dependencies
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install

# start the development server
npm run dev
```

### Editing

You can modify source files using your preferred IDE or editor. Changes will be visible immediately in the running dev server.

### Testing

Run the unit tests using:

```sh
npm run test
```

## Technologies

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
- Vitest for testing

## Deployment

Build the production bundle with:

```sh
npm run build
```

Then serve the contents of the `dist` directory with your hosting provider of choice (Netlify, Vercel, GitHub Pages, etc.).
